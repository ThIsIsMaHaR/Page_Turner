# 📚 PageTurner

**PageTurner** is a premium, aesthetic web novel platform built with the MERN stack. It offers a seamless experience for writers to publish their stories and for readers to enjoy them in a distraction-free, customizable environment.

---

## ✨ Features

### 🖋️ For Writers
- **Novel Management:** Create and manage your own web novels with ease.
- **Smart Publishing:** Auto-numbered chapter system with a distraction-free writing editor.
- **Author Dashboard:** Track your published works in a clean, professional layout.

### 📖 For Readers
- **Premium Reading Experience:** A minimalist reader with focus mode.
- **Customizable Themes:** Switch between **Dark**, **Light**, and **Sepia** modes to suit your eyes.
- **Dynamic Font Scaling:** Adjust font sizes for maximum comfort.
- **Persistent Progress:** Remembers where you left off (via LocalStorage integration).

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS, Axios, React Router
- **Backend:** Node.js, Express.js, MongoDB (Mongoose)
- **Auth:** JSON Web Tokens (JWT) & Bcryptjs
- **Icons:** Lucide React / FontAwesome
